Individual Staff Accounts

P0 — Must Ship
What This Feature Includes
  • Unique login credentials per staff member (email + password or magic link)
  • Profile fields: full name, display name, photo, bio, phone, email, job title
  • Specialties and certifications tagging
  • Employment type flag (full-time, part-time, contractor, booth renter)
  • Account status management (active, inactive, on leave)
Key User Flows
Admin Creates Account: Settings → Staff → Add New → Enter name, email, role → System sends activation link → Staff sets password & completes profile
Staff Edits Own Profile: My Profile → Edit fields (photo, bio, phone) → Save → Changes reflected on booking page if public
Admin Deactivates Staff: Staff list → Select member → Deactivate → All future appointments flagged for reassignment → Login disabled
How Top Platforms Handle This
Mangomint uses a clean staff profile with photo, bio, and specialties displayed on the booking page. Boulevard separates internal profile from client-facing profile. GlossGenius keeps it minimal — name, photo, services. Fresha includes employment type and tracks active/inactive status per team member.

Unlimited Staff Accounts

P0 — Must Ship
What This Feature Includes
  • No cap on number of staff accounts per location
  • Pricing model decision: flat fee vs. per-staff pricing tiers
  • Bulk invite capability for large teams (10+ staff)
  • Staff list management with search, filter, and sort
  • Archive vs. delete for departed staff (preserve history)
Key User Flows
Owner Reviews Team: Settings → Staff → See all members with status badges → Filter by role/status → Sort by name or date added
Bulk Invite: Staff → Invite Multiple → Upload CSV or enter emails → Assign default role → Send batch invitations
Archive Staff: Staff → Select departed member → Archive (not delete) → Historical appointments and revenue preserved → Name appears greyed in past records
How Top Platforms Handle This
Mangomint charges per provider (not assistants/front desk), making it truly unlimited for non-provider roles. Boulevard has per-location staff included in plan tier. GlossGenius is single-provider focused. Fresha allows unlimited team members on all plans with no per-staff fees.

Staff Invite & Onboarding Flow

P0 — Must Ship
What This Feature Includes
  • Email and/or SMS invite delivery with branded template
  • Guided setup wizard: accept invite → set password → choose role → configure availability → assign services
  • Invite link expiry and resend capability
  • Onboarding checklist tracking (profile complete, hours set, services assigned)
  • Bulk invite for multiple team members simultaneously
Key User Flows
Owner Sends Invite: Staff → Invite New → Enter name + email/phone → Select role template → Send → Staff receives branded email with 'Join [Business Name]' button
Staff Accepts & Sets Up: Click invite link → Create password → Upload photo → Set weekly hours → Review assigned services → Confirm → Dashboard loads
Owner Tracks Onboarding: Staff list shows onboarding status (Invited / Setup In Progress / Complete) → Nudge incomplete setups → Resend expired invites
How Top Platforms Handle This
Mangomint sends a clean email invite; new staff land on a guided setup page with hours and services. Boulevard includes an onboarding checklist visible to the admin. GlossGenius handles it through the owner's app (owner adds, then shares a code). Fresha uses email invites with step-by-step setup and requires owner approval before the profile goes live.

Pre-Set Role Templates

P0 — Must Ship
What This Feature Includes
  • Default templates: Owner (full access), Manager (most access minus billing settings), Service Provider (own schedule + clients), Assistant (limited booking actions), Front Desk (booking + checkout, no financials)
  • One-click role assignment during invite or post-setup
  • Template shows clear summary of what each role can/cannot access
  • Ability to duplicate and customize templates (bridges to P1 Custom Permission Groups)
  • Role displayed as badge on staff list and profile
Key User Flows
Assign Role During Invite: Invite flow → Select role from dropdown (Owner / Manager / Provider / Assistant / Front Desk) → Preview permissions summary → Confirm → Permissions auto-applied
Change Role: Staff profile → Edit Role → Select new template → Review permission diff → Confirm → Access updates immediately across all devices
View Role Permissions: Settings → Roles → Select template → See full permission matrix (Calendar ✓, Financials ✗, Client Data ✓, etc.)
How Top Platforms Handle This
Mangomint offers Owner, Manager, and Staff roles with clear permission matrices visible in settings. Boulevard has granular permission groups (6+ default roles) that map to salon hierarchies. GlossGenius is simpler — just owner vs. team member. Fresha provides Owner, Manager, and Employee defaults with 25+ individual permission toggles per role.

Service-to-Staff Assignment

P0 — Must Ship
What This Feature Includes
  • Link specific services to qualified staff members
  • Unassigned staff auto-excluded from booking flow for that service
  • Bulk assignment: assign staff to all services in a category at once
  • Per-service staff list visible in service settings
  • Assignment reflected in booking widget (only qualified providers shown)
Key User Flows
Assign from Service: Services → Select service → Staff tab → Check/uncheck providers who perform this service → Save → Booking widget updates immediately
Assign from Staff Profile: Staff → Select member → Services tab → Toggle services on/off → Save → Affects booking availability for that provider
Client Booking Experience: Client selects service → Only qualified providers shown → Selects provider → Only their available times displayed
How Top Platforms Handle This
Mangomint lets you assign services per team member from either the service or staff page, with a toggle matrix. Boulevard uses a similar two-way linking and also supports skill-level tagging (junior, senior) per service. GlossGenius auto-assigns all services to the provider. Fresha has a dedicated 'Team members' tab inside each service with checkboxes.

Staff-Specific Service Pricing

P0 — Must Ship
What This Feature Includes
  • Override base service price per individual staff member
  • Support for level-based pricing (Junior $50, Senior $70, Master $90)
  • Duration overrides per staff (senior may be faster/slower)
  • Price displayed to client during booking when selecting provider
  • Reporting: track revenue impact of price differentials
Key User Flows
Set Price Override: Staff → Select member → Services → Edit service → Override price and/or duration → Save → Base price unchanged for other staff
Level-Based Setup: Services → Select service → Pricing tab → Add pricing levels (Junior / Senior / Master) → Assign staff to levels → Prices auto-populate
Client Sees Pricing: Booking page → Select service → Choose provider → Price shown reflects that provider's rate → Confirmation shows provider-specific total
How Top Platforms Handle This
Mangomint supports per-staff pricing overrides on any service — you edit the price directly on the team member's service assignment. Boulevard offers pricing levels (e.g., Level 1–5) that group staff and set service prices by tier. GlossGenius does not currently support staff-specific pricing. Fresha allows individual price and duration overrides per team member per service.

Granular Permission Controls

P0 — Must Ship
What This Feature Includes
  • Permission categories: Calendar (view/edit own, view/edit all), Clients (view/edit/delete), Financials (view reports, process payments, issue refunds), Settings (business, services, staff), Booking (create/edit/cancel for self or others)
  • Toggle-level granularity per permission
  • Permission changes take effect immediately across web and mobile
  • Visual permission matrix for easy admin review
  • Permission-locked UI: hidden or greyed-out elements for unauthorized actions
Key User Flows
Admin Reviews Permissions: Settings → Roles & Permissions → Select role → See full toggle matrix organized by category → Toggle individual permissions on/off → Save
Staff Encounters Locked Feature: Staff clicks restricted area (e.g., Revenue Reports) → Sees 'You don't have access. Contact your manager.' → Feature hidden from navigation for cleaner UX
Emergency Access Grant: Owner → Staff profile → Temporarily grant elevated access → Set expiry (optional) → Staff can perform action → Access auto-reverts
How Top Platforms Handle This
Mangomint has ~15 permission toggles organized into clear categories (Appointments, Clients, Products, Reports, Settings). Boulevard offers the most granular system with 30+ individual permissions grouped by functional area. GlossGenius has minimal permissions — mostly owner vs. staff binary. Fresha provides ~25 toggles with clear on/off switches per team member.

Boulevard provides regular weekly hours plus 'schedule overrides' for specific date ranges. GlossGenius manages hours through the owner's app. Fresha offers a full weekly schedule editor with 15-minute increments and date-specific overrides.

Personal Time Blocks

P0 — Must Ship
What This Feature Includes
  • Block time on calendar for non-appointment activities (lunch, breaks, admin, training)
  • Recurring blocks (daily lunch 12–1) and one-off blocks
  • Block types/labels: Lunch, Break, Meeting, Personal, Training
  • Blocked time auto-prevents client bookings in that slot
  • Staff can manage own blocks (permission-gated); admin can manage all
Key User Flows
Staff Adds Block: Calendar → Click/tap empty slot → 'Block Time' → Select type (Lunch/Break/Personal) → Set start/end → Recurring? → Save → Slot greyed out on booking page
Admin Adds Team Block: Calendar → Select staff column → Block time → Label as 'Team Meeting' → Apply to multiple staff → Save → All affected calendars updated
Client Booking Around Blocks: Client picks service + provider → Blocked slots invisible/unavailable → Only open times shown → No visibility into why slot is blocked
How Top Platforms Handle This
Mangomint calls these 'Blocked Times' with configurable reasons and recurrence. Boulevard uses 'Personal Events' that sit on the calendar alongside appointments and can recur. GlossGenius has a simple 'Block off time' feature from the calendar. Fresha uses 'Blocked time' with type labels and supports both one-time and recurring blocks with an easy drag-to-create calendar UI.
